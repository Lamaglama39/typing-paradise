# typing-paradise
タイピングの楽園へようこそ、
<br>
このアプリは今までにないタイピング体験を提供します。
<br>
ゲーム開始と同時に四方八方から襲いかかる文字を、心ゆくまでお楽しみください。
<br>
<br>

# 特徴
- タイピング力だけでなく、動体視力が鍛えられます
- タイピング対象が複数あるため、自身でどれをタイピングするか選べます
- 文字サイズや色は常にランダムのため、飽きずに楽しめます

<br>
<br>
